# Kahicool website

Replacing the existing episerver powered public website and internal advice dispensing tool, AdviserNet, with a modern ruby-on-rails and Contentful powered web app.

Developing and maintaining a design system to ensure consistency of service across all Citizens Advice web sites and apps. Learning more about accessibility and ensuring the sites perform to the latest WCAG 2.1 AA standards.
